Photo Gallery

The first project using JavaScript and jQuery plugins, the goal of Project 4 was make an interactive photo gallery, opening thumbnails into a module with a light-box caption for each. Using the search field, photos that don't the title and captions were hidden and the only the images matching were displayed. Although I struggling somewhat with the JavaScript, it was a great project to put some of it to practice, and very satifying once complete.

Takeaway

The image gallery project really opened my eyes to how much work can actually go behind simple tasks that seem insanely easy for the end user. Displaying filtered results from typing a letter into a search bar, actually requires quite a bit of code and problem solving when you're just jumping into the coding side of things. I still have alot to learn about JavaScript, but this was an awesome introduction to the complex and underappreciated langauge.
